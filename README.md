# minerva-verify

A WIP script to verify all your files downloaded from Myrient.

## How to install

### Linux

Just run the bash file

```bash
curl -fsSL https://raw.githubusercontent.com/marshfellow42/minerva-verify/refs/heads/main/install/install.sh
```

Then run the command

```bash
minerva-verify
```

To update the app, just run the command below

```bash
minerva-verify update
```

## Configuration File

### Windows

- ${APPDATA}/minerva-verify/config.toml
